node-es6
========

A `node --harmony` executable with traceur compilation using `.traceurrc` via the [rc module](https://www.npmjs.org/package/rc).

#Install

```
$ npm install -g node-es6
```

#Run

```js
// index.js
let helloWorld = 'hello world'
console.log(hello)
```

```
$ node-es6 index.js
hello world
```
